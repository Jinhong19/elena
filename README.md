# elena

## Dependencies
Create and activate virtual environment  
`python3 -m venv venv`  
`source venv/bin/activate`

Install dependencies  
`pip3 install -r requirements.txt`

## Run Server Locally
`python3 Back-End/server.py`

## Open in browser
open in a browser: Fron-End/main.html
